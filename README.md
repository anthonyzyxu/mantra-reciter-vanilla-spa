# 咒語背誦訓練平台

> 從大悲咒開始，用認知科學的方法系統化掌握梵音咒語背誦。

---

## 一句話介紹

一個零依賴的純靜態網頁，將長篇梵音咒語拆解為結構化段落，配合七日四階段訓練計畫與多感官記憶技巧，幫助初學者從零開始逐步掌握全文背誦。

## 預覽

打開 `memory-plan.html` 即可看到：

- 🕉️ **金色梵音咒語展示**（帶呼吸光暈動畫）
- 📖 **五段文本結構分析**（含韻律模式亮點）
- 📜 **寫經風格全文卡片**（深底金字，模擬紺紙金泥）
- 🗓️ **四階段七日訓練計畫**（2×2 網格，含每日時間分配）
- 🧠 **七種核心記憶技巧**（間隔重複、睡眠固化、費曼法等）
- ⚠️ **四條注意事項**（避免常見學習誤區）

## 快速開始

```bash
# 唯一需要的指令：用瀏覽器打開
open memory-plan.html

# 或者啟動本地伺服器
python3 -m http.server 8080
# 打開 http://localhost:8080/memory-plan.html
```

無需 `npm install`、無需建置步驟。Clone 下來，雙擊打開，直接使用。

## 設計理念

配色取材自佛教寫經的物質文化：

- **紺紙**（深藍染紙）作底色
- **泥金**書法作文字
- **朱砂**印記作點綴

四個訓練階段使用漸進色編碼（青綠→琥珀→橙→紅），色彩本身即為記憶線索。

## 訓練計畫結構

| 階段 | 天數 | 目標 | 色標 |
|------|------|------|------|
| 第一階段 | Day 1–2 | 聆聽跟讀，建立聽覺記憶 | 青綠 |
| 第二階段 | Day 3–4 | 分段背誦，理解結構 | 琥珀 |
| 第三階段 | Day 5–6 | 串聯強化，干擾訓練 | 暖橙 |
| 第四階段 | Day 7 | 獨立背誦，完成驗收 | 朱紅 |

## 記憶方法論

訓練計畫參考以下認知科學原理：

- 間隔重複（Spaced Repetition）
- 睡眠記憶鞏固（Sleep-dependent Memory Consolidation）
- 費曼學習法（Feynman Technique）
- 雙重編碼理論（Dual Coding Theory）
- 多感官學習（Multi-sensory Learning）
- 干擾效應訓練（Contextual Interference）

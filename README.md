# 梵音咒語記憶訓練平台

> 大悲咒 · 楞嚴咒 — 用認知科學的方法系統化掌握梵音咒語背誦。

---

## 一句話介紹

一個零依賴的純靜態網頁平台，將長篇梵音咒語拆解為結構化段落，配合多階段訓練計畫與多感官記憶技巧，幫助初學者從零開始逐步掌握全文背誦。

## 快速開始

```bash
# 入口首頁
open index.html

# 或啟動本地伺服器（推薦，確保頁面間導航正常）
python3 -m http.server 8080
# 打開 http://localhost:8080
```

無需 `npm install`、無需建置步驟。Clone 下來，雙擊打開，直接使用。

## 頁面一覽

### 入口首頁

微軟磁貼（Metro）風格，兩塊大色塊選擇磚，點擊進入對應訓練頁。

| 磁貼 | 咒語 | 行數 | 訓練週期 |
|------|------|------|----------|
| 🪷 青綠 | 大悲咒 | 14 行 | 7 日 |
| ⚡ 深藍 | 楞嚴咒 | 193 行 | 21 日 |

### 訓練頁面

每個訓練頁面包含：

- 🕉️ **金色梵音 Hero**（呼吸光暈動畫）
- 📖 **五段文本結構分析**（總覽表 + 記憶錨點亮點卡）
- 📜 **寫經風格全文卡片**（深底金字，每句獨立一行）
- 🗓️ **四階段訓練計畫**（2×2 網格，含每日時間分配）
- 🧠 **七種核心記憶技巧**（咒語特定的記憶方法）
- ⚠️ **四條注意事項**（避免常見學習誤區）
- 🔗 **YouTube 音頻參考**（Phase 1 跟讀用）
- ↩️ **返回首頁**（左上角固定導航）

## 設計理念

配色取材自佛教寫經的物質文化：

- **紺紙**（深藍染紙）作底色
- **泥金**書法作文字
- **朱砂**印記作點綴

四個訓練階段使用漸進色編碼（青綠→琥珀→橙→紅），色彩本身即為記憶線索。

## 記憶方法論

訓練計畫參考以下認知科學原理：

- 間隔重複（Spaced Repetition）
- 睡眠記憶鞏固（Sleep-dependent Memory Consolidation）
- 費曼學習法（Feynman Technique）
- 雙重編碼理論（Dual Coding Theory）
- 多感官學習（Multi-sensory Learning）
- 干擾效應訓練（Contextual Interference）
